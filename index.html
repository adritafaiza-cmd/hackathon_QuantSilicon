Every problem found.

01
Shared fxp_pkg.sv Package
Centralized Q16.16 arithmetic utilities ‚Äî fxp_mul_q16, fxp_abs, shared constants ‚Äî eliminating scaling mismatches across modules.
‚úì Consistent arithmetic
02
Feature Engine Backpressure
Output register buffering + valid/ready handshake holds output stable until downstream is ready. Input only accepted when safe to overwrite.
‚úì Backpressure-safe
03
Signal Engine Simplification
Direct combinational expression for signal computation. Eliminated intermediate registers, reducing combinational depth for cleaner synthesis.
‚úì Reduced logic depth
04
Risk Engine Safety Logic
Registered kill-switch output prevents glitches. Absolute position via fxp_abs ensures stable exposure evaluation under all pipeline conditions.
‚úì Glitch-free kill logic
05
Branch Synchronization
Dual independent buffers ‚Äî one per branch ‚Äî with out_valid asserted only when both are ready. Guarantees aligned signal + risk output pairs.
‚úì Zero mismatched pairs
06
Pop-Based Buffer Clearing
Buffers dequeue only on successful downstream consume (pop = out_valid && out_ready). Prevents data loss under arbitrary downstream stalls.
‚úì Zero data loss
